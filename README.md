

# 章鱼扭动效果（squirmy-octopus-ui）

用 p5.js 实现的章鱼触手扭动动画特效

## 在线预览


## 如何使用

1. 克隆本仓库：  
   ```bash
   git clone https://github.com/gpboyer1/squirmy-octopus-ui.git
   ```
2. 双击打开 `main.html` 文件，即可在浏览器中看到章鱼扭动动画。

## 主要文件

- `main.html`：包含全部动画代码和 p5.js 引用。

## 效果说明

本项目通过 p5.js 绘制章鱼触手的动态扭动效果，适合用于前端动画、视觉实验等场景。动画通过极坐标和三角函数模拟章鱼触手的自然摆动，画面简洁而富有动感。
